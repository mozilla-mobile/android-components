fun initialize(context: Context, useShippedDomains: Boolean = true, useCustomDomains: Boolean = false, loadDomainsFromDisk: Boolean = true): Unit (source)

Initializes this provider instance by making sure the shipped and/or custom domains are loaded.

Parameters

context - the application context

useShippedDomains - true (default) if the domains provided by this module should be used, otherwise false.

useCustomDomains - true if the custom domains provided by CustomDomains should be used, otherwise false (default).

loadDomainsFromDisk - true (default) if domains should be loaded, otherwise false. This parameter is for testing purposes only.
