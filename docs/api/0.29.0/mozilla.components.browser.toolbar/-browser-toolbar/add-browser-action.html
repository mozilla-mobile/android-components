addBrowserAction

fun addBrowserAction(action: Action): Unit (source)

Overrides Toolbar.addBrowserAction

Adds an action to be displayed on the right side of the toolbar (outside of the URL bounding box) in display mode.

If there is not enough room to show all icons then some icons may be moved to an overflow menu.
