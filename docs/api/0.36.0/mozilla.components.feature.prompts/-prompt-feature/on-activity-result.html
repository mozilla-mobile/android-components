onActivityResult

fun onActivityResult(requestCode: Int, resultCode: Int, intent: Intent?): Unit (source)

Forward the calls to onActivityResult on your Activity or Fragment, to let the feature know, the results of intents for handling prompt requests, that need to be performed by other apps like file chooser requests.

Parameters

requestCode - The code of the app that requested the intent.

intent - The result of the request.
