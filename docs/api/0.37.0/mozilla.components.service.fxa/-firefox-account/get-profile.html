fun getProfile(ignoreCache: Boolean): Deferred<Profile> (source)

Fetches the profile object for the current client either from the existing cached account, or from the server (requires the client to have access to the profile scope).

Parameters

ignoreCache - Fetch the profile information directly from the server

Exceptions

Unauthorized - We couldn't find any suitable access token to make that call. The caller should then start the OAuth Flow again with the "profile" scope.

Return
Deferred<Profile> representing the user's basic profile info
