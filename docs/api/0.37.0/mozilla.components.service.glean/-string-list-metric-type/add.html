fun add(value: String): Unit (source)

Appends a string value to one or more string list metric stores. If the string exceeds the maximum string length, defined as MAX_STRING_LENGTH, it will be truncated.

If adding the string to the lists would exceed the maximum value defined as StringListsStorageEngine.MAX_LIST_LENGTH_VALUE, then the storage engine will drop the new value and it will not be added to the list.

Parameters

value - This is a user defined string value. The maximum length of this string is MAX_STRING_LENGTH.
