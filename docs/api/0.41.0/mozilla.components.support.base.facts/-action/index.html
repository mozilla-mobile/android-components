Action

enum class Action (source)

A user or system action that causes Fact instances to be emitted.

Enum Values

CLICK

The user has clicked on something.

TOGGLE

The user has toggled something.

COMMIT

The user has committed an input (e.g. entered text into an input field and then pressed enter).

INTERACTION

A generic interaction that can be caused by a previous action (e.g. the user clicks on a button which causes a Fact with CLICK action to be emitted. This click may causes something to load which emits a follow-up a Fact with INTERACTION action.
