getPublicSuffix

fun getPublicSuffix(domain: String): <ERROR CLASS> (source)

Returns the public suffix of the given domain. Returns null if the domain is a public suffix itself.

E.g.:

wwww.mozilla.org -> org
www.bcc.co.uk    -> co.uk
a.b.ide.kyoto.jp -> ide.kyoto.jp
