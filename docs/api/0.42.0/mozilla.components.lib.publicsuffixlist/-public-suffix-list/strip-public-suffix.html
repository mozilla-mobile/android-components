stripPublicSuffix

fun stripPublicSuffix(domain: String): <ERROR CLASS> (source)

Strips the public suffix from the given domain. Returns the original domain if no public suffix could be stripped.

E.g.:

wwww.mozilla.org -> www.mozilla
www.bcc.co.uk    -> www.bbc
a.b.ide.kyoto.jp -> a.b
