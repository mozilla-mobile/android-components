fun set(value: List<String>): Unit (source)

Sets a string list to one or more metric stores.

Parameters

value - This is a user defined string list. The maximum length of each string in the list is defined by MAX_STRING_LENGTH, while the maximum length of the list itself is defined by StringListsStorageEngine.MAX_LIST_LENGTH_VALUE. If a longer list is passed into this function, then the additional values will be dropped from the list and the list, up to the StringListsStorageEngine.MAX_LIST_LENGTH_VALUE, will still be recorded.
