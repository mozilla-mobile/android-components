withRoundedCorners

@CheckResult fun Bitmap.withRoundedCorners(cornerRadiusPx: Float): Bitmap (source)

Returns a new bitmap that is the receiver Bitmap with four rounded corners; the receiver is unmodified.

This operation is expensive: it requires allocating an identical Bitmap and copying all of the Bitmap's pixels. Consider these theoretically cheaper alternatives:

  • android:background= a drawable with rounded corners
  • Wrap your bitmap's ImageView with a layout that masks your view with rounded corners (e.g. CardView)
