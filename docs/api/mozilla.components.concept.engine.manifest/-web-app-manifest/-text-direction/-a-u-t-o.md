# AUTO

`AUTO` [(source)](https://github.com/mozilla-mobile/android-components/blob/master/components/concept/engine/src/main/java/mozilla/components/concept/engine/manifest/WebAppManifest.kt#L165)

If the value is set to auto, the browser will use the Unicode bidirectional algorithm to make a best guess
about the text's direction.
