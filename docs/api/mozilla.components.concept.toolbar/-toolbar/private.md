# private

`abstract var private: `[`Boolean`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-boolean/index.html) [(source)](https://github.com/mozilla-mobile/android-components/blob/master/components/concept/toolbar/src/main/java/mozilla/components/concept/toolbar/Toolbar.kt#L43)

Sets/gets private mode.

In private mode the IME should not update any personalized data such as typing history and personalized language
model based on what the user typed.
