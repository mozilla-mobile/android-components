# &lt;init&gt;

`AddonsManagerAdapter(addonCollectionProvider: `[`AddonCollectionProvider`](../../mozilla.components.feature.addons.amo/-addon-collection-provider/index.md)`, addonsManagerDelegate: `[`AddonsManagerAdapterDelegate`](../-addons-manager-adapter-delegate/index.md)`, addons: `[`List`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin.collections/-list/index.html)`<`[`Addon`](../../mozilla.components.feature.addons/-addon/index.md)`>)`

An adapter for displaying add-on items. This will display information related to the state of
an add-on such as recommended, unsupported or installed. In addition, it will perform actions
such as installing an add-on.
