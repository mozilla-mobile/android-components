# add

`fun add(value: `[`String`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-string/index.html)`): `[`Unit`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-unit/index.html) [(source)](https://github.com/mozilla-mobile/android-components/blob/master/components/service/glean/src/main/java/mozilla/components/service/glean/private/StringListMetricType.kt#L42)

Appends a string value to one or more string list metric stores.  If the string exceeds the
maximum string length or if the list exceeds the maximum length it will be truncated.

### Parameters

`value` - This is a user defined string value. The maximum length of
