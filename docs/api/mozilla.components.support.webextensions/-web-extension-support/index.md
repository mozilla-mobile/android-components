# WebExtensionSupport

`object WebExtensionSupport` [(source)](https://github.com/mozilla-mobile/android-components/blob/master/components/support/webextensions/src/main/java/mozilla/components/support/webextensions/WebExtensionSupport.kt#L30)

[Engine](../../mozilla.components.concept.engine/-engine/index.md) are reflected in the browser state by dispatching the
